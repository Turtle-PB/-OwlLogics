📥 Quick Start

Option 1: Live Demo (Easiest)

Visit https://turtle-pb.github.io/OwlLogic/ — no download needed.

Option 2: Download & Run

  1. Clone: git clone https://github.com/Turtle-PB/OwlLogic.git
  2. Open index.html in any modern browser
  3. Done — the app boots and MSQM starts streaming automatically

Option 3: Local Server (for PWA/mobile)

cd OwlLogic
python -m http.server 8765
# Open http://localhost:8765 on any device

Option 4: Install as PWA on Phone

  • Android: Chrome → 3-dot menu → Install app
  • iOS: Safari → Share → Add to Home Screen
  • Desktop: Chrome/Edge → install icon in address bar
